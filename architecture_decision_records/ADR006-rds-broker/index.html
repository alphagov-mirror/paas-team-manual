ADR006: RDS broker

Context

We need to provide tenants with the ability to provision databases for use in their applications. Our first iteration of this will be using RDS. We investigated some implementations of a service broker which supported RDS

  • cf platform eng
  • 18F

Decision

We will use the cf platform eng rds broker. As this is not a supported product, we will fork this and maintain this and implement new features ourselves.

Status

Accepted

Consequences

We will be maintaining a new service broker, but have a head start on creating it by basing it on an existing service broker.
